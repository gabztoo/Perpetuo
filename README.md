# Perpetuo AI Gateway

A production-grade AI Gateway sitting between your applications and LLM providers.

## Features
- **Unified API**: OpenAI-compatible endpoint.
- **Reliability**: Deterministic fallbacks, retries, and circuit breakers.
- **Observability**: Built-in metrics, logs, and traces.
- **Cost Control**: Smart routing based on cost policies.

## Setup
1. `pnpm install`
2. `docker-compose up -d` (for dependencies)
3. `pnpm dev`
